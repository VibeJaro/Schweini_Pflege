# Schweini_Pflege

Kleine Browser-App (ohne Build-Tooling): Schweinchen waschen, spülen, föhnen.

## Struktur

- `index.html` – Markup und SVG-Layer
- `styles_main.css` – UI- und Animations-Styles
- `game.js` – Spiel-Loop, Input, Partikel, UI-Logik
- `audio.js` – AudioManager für One-Shots + kontinuierliche Werkzeug-Sounds

## Start

Datei direkt im Browser öffnen oder lokal per Static Server starten.
